# 171-23 guruh talabasi Do'stbekova Charos tomonidan qilingan!

Ishlatish tartibi:
1. Avval muhit yaratib olinadi: *python -m venv Charos*

2. Muhitga kirib kutubxonalar yuklab olianadi:
   linuxda: *source Charos/bin/activate*  =>   *pip install -r requirements*
   windowsda: *cmd Charos/scripts/activate*   =>  *pip install -r requirements*

3. main.py ishga tushiriladi:  *streamlit run main.py*
